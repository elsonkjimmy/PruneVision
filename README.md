## 🚀 About PruneVision

PruneVision is an innovative agriculture technology (AgTech) application that leverages computer vision and machine learning to assess crop quality, specifically focused on prune quality evaluation. Built with React, TypeScript, and Vite, it demonstrates how modern web technologies can be applied to solve real-world agricultural challenges.

### Key Features

- 🌱 **Crop Quality Assessment** - AI-powered analysis of prune quality using computer vision
- 📊 **Visual Analysis Dashboard** - Interactive interface for examining crop characteristics
- 🎯 **Real-time Processing** - Fast image processing and quality scoring
- 📱 **Responsive Design** - Works seamlessly on desktop and mobile devices
- 🔬 **Detailed Metrics** - Comprehensive quality metrics and classifications

## 🚀 Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn
- Modern web browser

### Installation

1. **Clone the repository**
```bash
git clone https://github.com/elsonkjimmy/PruneVision.git
cd PruneVision
```

2. **Install dependencies**
```bash
npm install
# or
yarn install
```

3. **Start the development server**
```bash
npm run dev
# or
yarn dev
```

4. **Open in browser**
Navigate to `http://localhost:5173`
